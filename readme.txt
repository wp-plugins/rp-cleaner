RP cleaner is a kind of plugin which gives user to beautify his / her dashboard as per his / her needs.

== Description ==

RP cleaner allows you to clean up your dashboard and add a welcome message for you with some custom text and let you do a clean work in wordpress.

We'll keep updating this plugin in regular basis and gives you a good control in your wordpress dashboard.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'rpcleaner'
3. Activate rpcleaner from your Plugins page. 

= From WordPress.org =

1. Download rpcleaner.
2. Upload the 'rpcleaner' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate rpcleaner from your Plugins page.
